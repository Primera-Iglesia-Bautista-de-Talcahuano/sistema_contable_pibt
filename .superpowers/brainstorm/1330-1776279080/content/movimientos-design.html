Design decisions

  • Mobile: card list — each movement is a tap target with icon, label, meta, amount and status. Cancelled rows are dimmed + strikethrough.
  • Mobile: filter chips — active filters shown as dismissible pills below the search bar. Easy to see what's active and remove it.
  • Desktop: full table with sortable columns, category, pagination at bottom.
  • Search bar always visible on both — no need to open a separate modal to start filtering.
  • Clicking any row → detail page (not a modal). Keeps navigation predictable and shareable by URL.
