Design decisions

  • Usuarios: Avatar initials + inline name edit + role badge + toggle switch per row. "Guardar" appears faded — becomes visible on hover/change.
  • Role badges are color-coded — green for Admin, purple for Operator, grey for Viewer. Instantly scannable.
  • Inactive users are dimmed at 60% opacity — present but visually suppressed.
  • Configuración: Entity badges are color-coded by type — green for movements, purple for users, red for cancellations. Action column is bold.
  • Alternating row tints (very subtle) on both tables aid readability without heavy grid lines.
  • Mobile (not shown): Usuarios becomes a card list with each user as a card. Config becomes a compact list with date + action + user on each row.
